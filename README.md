# ATM Simulator using Switch Case in C

I wrote a program for ATM simulator using switch case in C language. This console-based application simulates basic ATM operations with a simple menu-driven interface.

---

## 📌 About This Program

I created this ATM simulator program using **switch case** statements to handle different banking operations. The program allows users to perform basic transactions through a simple menu system.

---

## 🔧 What I Used

- **Language**: C
- **Concept**: Switch Case Statements
- **Features**: PIN Authentication, Balance Check, Deposit, Withdraw

---

## 💡 How It Works

The program uses **switch case** to handle user choices:

```c
switch (choice) {
    case 1:  // Check Balance
    case 2:  // Deposit
    case 3:  // Withdraw
    case 4:  // Exit
    default: // Invalid Choice
}
